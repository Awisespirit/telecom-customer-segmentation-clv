# 📊 Telecom Customer Segmentation using CLV

## Business Problem

NexaSat needed to optimize marketing spend across a diverse customer base.

## Objective

Segment customers based on lifetime value to improve targeting and revenue.

## Tools

SQL | Python | Excel | Power BI

## Process

* Data cleaning and preparation
* Exploratory Data Analysis
* CLV calculation
* Customer segmentation

## Key Insights

* Top customers generated majority of revenue
* Mid-tier customers had growth potential

## Recommendations

* Focus marketing on high-value segments
* Improve retention strategies

## Outcome

Better marketing efficiency and improved decision-making
